### `MONGO_HOST`

*Optional*, default value: `localhost`

MongoDB host

### `MONGO_PORT`

*Optional*, default value: `27017`

MongoDB port

### `MONGO_USERNAME`

*Optional*, default value: ``

MongoDB username

### `MONGO_PASSWORD`

*Optional*, default value: ``

MongoDB password

### `MONGO_DATABASE`

*Optional*, default value: `brickserver`

MongoDB database

### `MONGO_SCHEMA`

*Optional*, default value: `mongodb`

MongoDB schema (do not modify this)
